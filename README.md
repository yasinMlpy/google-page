# Google Homepage Clone

A simple project that recreates the Google homepage using HTML and CSS.

## ✨ Features

- Centered Google logo
- Search input with rounded corners
- "Google Search" and "I'm Feeling Lucky" buttons
- Responsive layout using Flexbox

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox

## 📂 Project Structure

```
project/
│
├── index.html
├── google.css
└── google-1200-2.webp
```

## 🚀 Getting Started

1. Download or clone the repository.
2. Open `index.html` in your browser.

## 📸 Preview

This project includes:

- A centered Google logo
- A search bar
- Two action buttons below the search bar
